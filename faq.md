# Частые вопросы

Здесь собраны ответы на самые популярные вопросы о Harvi Code.

---

## Общие вопросы

### Что такое Harvi Code?

Harvi Code - это AI-помощник для разработки со встроенными AI-моделями. Он помогает в написании кода, отладке и автоматизации задач.

### Как работает Harvi Code?

Harvi Code использует встроенные AI-модели для понимания твоих запросов и превращения их в действия. Он может:

- Читать и писать файлы в твоем проекте
- Выполнять команды в терминале
- Использовать внешние инструменты через Model Context Protocol (MCP)

Ты взаимодействуешь с Harvi Code через интерфейс чата в VS Code.

### Что умеет Harvi Code?

Harvi Code помогает с разными задачами разработки:

* Генерация кода по описанию
* Рефакторинг существующего кода
* Исправление багов
* Написание документации
* Объяснение кода
* Ответы на вопросы о твоей кодовой базе
* Автоматизация рутинных задач
* Создание новых файлов и проектов

### Harvi Code бесплатный?

Информацию о ценах и лицензировании смотри на официальном сайте Harvi Code.

### Какие риски при использовании Harvi Code?

Harvi Code - мощный инструмент, и важно использовать его ответственно:

* **Harvi Code может ошибаться.** Всегда проверяй предложенные изменения перед одобрением.
* **Harvi Code может выполнять команды.** Будь осторожен с разрешением на выполнение команд, особенно с авто-подтверждением.

## Использование

### Как начать новую задачу?

Открой панель Harvi Code и напиши задачу в поле чата. Будь четким и конкретным в описании того, что хочешь сделать. Смотри [как писать запросы](/basic-usage/writing-requests) для лучших практик.

### Что такое режимы в Harvi Code?

[Режимы](/basic-usage/modes) - это разные персоны, которые может принимать Harvi Code, каждая со своим фокусом и набором возможностей. Встроенные режимы:

* **Code:** Для общих задач программирования
* **Architect:** Для планирования и технического лидерства
* **Ask:** Для ответов на вопросы и предоставления информации
* **Debug:** Для систематической диагностики проблем

Ты также можешь создавать [кастомные режимы](/features/custom-modes).

### Как переключаться между режимами?

Используй выпадающее меню в области ввода чата для выбора другого режима, или используй команду `/` для переключения на конкретный режим.

### Что такое инструменты и как их использовать?

[Инструменты](/basic-usage/tools) - это способ взаимодействия Harvi Code с твоей системой. Harvi Code автоматически выбирает и использует подходящие инструменты для выполнения задач. Тебе не нужно вызывать инструменты напрямую. Тебя попросят одобрить или отклонить каждое использование инструмента.

### Что такое упоминания контекста?

[Упоминания контекста](/basic-usage/context-mentions) - это способ предоставить Harvi Code конкретную информацию о твоем проекте, такую как файлы, папки или проблемы. Используй символ "@" с последующим элементом, который хочешь упомянуть (например, `@/src/file.ts`, `@problems`).

### Может ли Harvi Code запускать команды в терминале?

Да, Harvi Code может выполнять команды в терминале VS Code. Тебя попросят одобрить каждую команду перед выполнением, если не включено авто-подтверждение. Будь крайне осторожен с авто-подтверждением команд.

### Как давать обратную связь Harvi Code?

Ты можешь давать обратную связь, одобряя или отклоняя предложенные действия Harvi Code. Можешь добавить дополнительную обратную связь через поле для комментариев.

### Можно ли кастомизировать поведение Harvi Code?

Да, ты можешь кастомизировать Harvi Code несколькими способами:

* **Кастомные инструкции:** Предоставь общие инструкции для всех режимов или специфичные для режима
* **Кастомные режимы:** Создай свои режимы с настроенными промптами и правами на инструменты
* **`.roorules` файлы:** Создай `.roorules` файлы в проекте для дополнительных руководств
* **Настройки:** Настрой различные параметры, такие как авто-подтверждение, редактирование diff и другое

## Решение проблем

### Harvi Code не отвечает. Что делать?

* Проверь интернет-соединение
* Попробуй перезапустить VS Code
* Если проблема сохраняется, обратись в службу поддержки Harvi Code

### Я вижу сообщение об ошибке. Что оно означает?

Сообщение об ошибке должно содержать информацию о проблеме. Если не уверен, как ее решить, обратись в службу поддержки.

### Harvi Code сделал изменения, которые я не хотел. Как отменить?

Harvi Code использует встроенные возможности редактирования файлов VS Code. Ты можешь использовать стандартную команду "Undo" (Ctrl/Cmd + Z) для отмены изменений.

### Как сообщить об ошибке или предложить функцию?

Обратись в службу поддержки Harvi Code с описанием проблемы или предложением.
