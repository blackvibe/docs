# Интерфейс чата

Интерфейс чата - это твое главное окно для работы с Harvi Code. Он находится в панели Harvi Code, которую можно открыть, кликнув на иконку в боковой панели VS Code.

---

## Элементы интерфейса

Вот из чего состоит интерфейс чата:

1. **История чата** - здесь отображается вся переписка между тобой и Harvi Code. Твои запросы, ответы Harvi Code и все выполненные действия (редактирование файлов, команды и т.д.).

2. **Поле ввода** - сюда ты пишешь свои задачи и вопросы. Используй обычный язык, никаких специальных команд.

3. **Кнопки действий** - появляются над полем ввода и позволяют одобрить или отклонить предложенные Harvi Code действия. Набор кнопок меняется в зависимости от контекста.

4. **Кнопка отправки** - выглядит как самолетик, находится справа от поля ввода. Отправляет сообщение Harvi Code.

5. **Кнопка "плюс"** - находится вверху в шапке. Переключает на вкладку чата и фокусирует поле ввода. Чтобы начать новую сессию или очистить текущую задачу, используй эту кнопку.

6. **Кнопка настроек** - иконка шестеренки для открытия настроек и кастомизации поведения Harvi Code.

7. **Выбор режима** - выпадающий список слева от поля ввода. Здесь выбираешь, в каком режиме должен работать Harvi Code. Шестеренка рядом открывает настройки режимов.

## Совет: используй вторую боковую панель

Для удобства можешь перетащить Harvi Code во [вторую боковую панель](https://code.visualstudio.com/api/ux-guidelines/sidebars#secondary-sidebar) VS Code. Так ты сможешь держать Harvi Code открытым и одновременно пользоваться Explorer, Search и другими панелями.

Как это сделать:
1. Зажми иконку Harvi Code в боковой панели
2. Перетащи ее на правую сторону редактора
3. Теперь у тебя две боковые панели!

Больше советов по продуктивности в разделе [Советы и трюки](/tips-and-tricks).

## Работа с сообщениями

* **Кликабельные ссылки** - пути к файлам, URL и другие упоминания в чате кликабельны. Клик по пути откроет файл в редакторе, клик по URL - в браузере.

* **Копирование текста** - выдели текст и используй стандартное копирование (Ctrl/Cmd + C). У блоков кода есть отдельная кнопка "Copy".

* **Сворачивание сообщений** - кликни на сообщение, чтобы свернуть или развернуть его.

---

## Индикаторы статуса

* **Спиннер загрузки** - когда Harvi Code обрабатывает запрос, увидишь анимацию загрузки.
* **Сообщения об ошибках** - ошибки отображаются красным цветом.
* **Сообщения об успехе** - успешное выполнение показывается зеленым.
