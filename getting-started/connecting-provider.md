# Подключение AI провайдера

Harvi Code нужен доступ к языковой модели, чтобы работать. Сам по себе он не содержит AI - он использует внешние сервисы (провайдеры), которые предоставляют доступ к моделям.

Лучший вариант для старта - **Claude Sonnet 4.5**. Эта модель мощная, относительно недорогая и отлично работает "из коробки".

## Выбор провайдера

Вот несколько популярных вариантов:

- **Roo Code Router** - самый простой способ. Не нужен API ключ, просто подключи аккаунт. [Подробнее о Router](/roo-code-router/overview)

- **OpenRouter** - доступ к множеству моделей через один API ключ. Хороший баланс между простотой и гибкостью. [Как получить ключ](/providers/openrouter)

- **Anthropic** - прямой доступ к моделям Claude. Требует одобрения и может иметь [лимиты по запросам](https://docs.anthropic.com/en/api/rate-limits). [Как получить ключ](/providers/anthropic)

Harvi Code поддерживает [много других провайдеров](/providers) - можешь экспериментировать и выбирать, что тебе больше подходит.

:::tip Совет по выбору модели
Мы рекомендуем **Claude Sonnet 4.5**, потому что она просто работает. Можно попробовать и другие модели, но это добавит сложности - разные модели по-разному понимают инструкции, работают с инструментами и держат контекст. Если хочешь экспериментировать, выбирай модели, заточенные под структурированное мышление и работу с инструментами.
:::

## Настройка в VS Code

1. Открой панель Harvi Code (кликни на иконку в боковой панели)
2. На экране приветствия выбери своего провайдера
3. Если выбрал Roo Code Router - просто подключи аккаунт и выбери модель `anthropic/claude-sonnet-4-5`
4. Если выбрал другого провайдера - вставь API ключ в соответствующее поле
5. Выбери модель (ищи `claude-sonnet-4-5` или `anthropic/claude-sonnet-4-5`)
6. Готово!

Теперь можешь начинать работу с Harvi Code.
