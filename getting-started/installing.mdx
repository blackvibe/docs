Harvi Code - это расширение для VS Code, которое добавляет AI-помощника прямо в твой редактор. Установить его можно несколькими способами.

## Установка из VS Code Marketplace

Это самый быстрый способ:

1. Открой VS Code
2. Нажми на иконку расширений в боковой панели (или `Ctrl+Shift+X` / `Cmd+Shift+X`)
3. Введи в поиске "Harvi Code"
4. Найди расширение Harvi Code и нажми **Install**
5. Перезагрузи VS Code, если попросит

После установки в боковой панели появится иконка Harvi Code - кликни на нее, чтобы открыть панель.

## Установка из Open VSX Registry

Для редакторов без доступа к VS Code Marketplace (например, VSCodium):

1. Открой свой редактор
2. Перейди в раздел расширений
3. Найди "Harvi Code"
4. Установи расширение Harvi Code
5. Перезагрузи редактор

## Установка из VSIX файла

Если хочешь установить конкретную версию или сохранить установочный файл:

1. Найди Harvi Code в поиске расширений VS Code
2. Кликни правой кнопкой мыши на расширении
3. Выбери "Download Extension VSIX" для скачивания файла
4. Для установки: кликни на меню "..." в разделе расширений
5. Выбери "Install from VSIX..." и укажи путь к скачанному файлу

## Решение проблем

**Расширение не появилось**
- Перезапусти VS Code
- Проверь, что расширение включено в списке
- Попробуй отключить и включить его снова
- Посмотри логи в панели Output (View → Output, выбери "Harvi Code")

**Проблемы с установкой**
- Проверь интернет-соединение
- Убедись, что у тебя VS Code версии 1.84.0 или новее
- Если Marketplace недоступен, попробуй Open VSX Registry
