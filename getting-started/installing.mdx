# Установка Harvi

Harvi - это расширение для VS Code, которое добавляет AI-помощника прямо в твой редактор. Установить его можно несколькими способами:

1. **VS Code Marketplace** - самый простой способ для VS Code и Cursor
2. **Open VSX Registry** - для VSCodium и других совместимых редакторов
3. **Вручную из VSIX** - если нужна конкретная версия

---

## Установка из VS Code Marketplace

Это самый быстрый способ:

1. Открой VS Code
2. Нажми на иконку расширений в боковой панели (или `Ctrl+Shift+X` / `Cmd+Shift+X`)
3. Введи в поиске "Roo Code"
4. Найди расширение от RooVeterinaryInc и нажми **Install**
5. Перезагрузи VS Code, если попросит

После установки в боковой панели появится иконка Harvi - кликни на нее, чтобы открыть панель.

---

## Установка из Open VSX Registry

Для редакторов без доступа к VS Code Marketplace (например, VSCodium):

1. Открой свой редактор
2. Перейди в раздел расширений
3. Найди "Roo Code"
4. Установи расширение от RooVeterinaryInc
5. Перезагрузи редактор

---

## Установка из VSIX файла

Если хочешь установить конкретную версию:

1. **Скачай VSIX файл:**
   - Найди нужный релиз на [GitHub Releases](https://github.com/RooCodeInc/Roo-Code/releases)
   - Скачай файл с расширением `.vsix`

2. **Установи в VS Code:**
   - Открой VS Code
   - Перейди в раздел расширений
   - Кликни на меню "..." вверху
   - Выбери "Install from VSIX..."
   - Укажи путь к скачанному файлу

## Решение проблем

**Расширение не появилось**
- Перезапусти VS Code
- Проверь, что расширение включено в списке
- Попробуй отключить и включить его снова
- Посмотри логи в панели Output (View → Output, выбери "Roo Code")

**Проблемы с установкой**
- Проверь интернет-соединение
- Убедись, что у тебя VS Code версии 1.84.0 или новее
- Если Marketplace недоступен, попробуй Open VSX Registry
